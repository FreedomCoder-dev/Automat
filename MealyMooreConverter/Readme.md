# MealyMooreConverter

## Информация о системе

### Архитектура процессора
- Архитектура: x86_64
- Размер шины: 64 бита

### Операционная система
- ОС: Ubuntu 20.04 LTS
- Версия: 20.04

### Зависимости
- CMake: 3.10 или выше
  - Ссылка на загрузку: https://cmake.org/download/
  - Путь include/src: Устанавливается в систему

- GCC: 9.3.0 или выше
  - Ссылка на загрузку: https://gcc.gnu.org/releases.html
  - Путь include/src: Устанавливается в систему
